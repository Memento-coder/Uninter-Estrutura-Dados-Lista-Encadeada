# Estrutura de Dados - Lista Encadeada

Trabalho da disciplina de Estrutura de Dados / Programação III.

## Questão 1
Implementação de uma lista encadeada simples para simular uma fila de atendimento hospitalar com prioridade.

Regras:
- Pacientes com cartão amarelo (A) têm prioridade sobre os verdes (V)
- Numeração automática:
  - Verde inicia em 1
  - Amarelo inicia em 201

Funcionalidades:
- Inserir paciente
- Mostrar lista de espera
- Atender paciente

## Execução

python fila_prioridade.py
